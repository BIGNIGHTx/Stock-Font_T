# SomBoon Electric - Stock Management System

## Overview
A stock management web application for SomBoon Electric shop.

## Features
- Dashboard: Overview of sales, inventory, and reports
- Inventory: Manage product stock, add/edit/delete items
- Sales: Record and track sales transactions
- Reports: View sales and inventory reports
- Settings: Toggle dark/light mode, manage preferences

## Tech Stack
- React + Vite
- Tailwind CSS v4
- React Router

## Expected Behavior
- Dark mode toggle should switch entire app theme
- Navigation between pages should work correctly
- Data should display correctly in all pages
